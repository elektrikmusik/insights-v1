# Vibe Coding Workflow

This is the consolidated hub for the Vibe Coding Workflow. Choose a step to begin or continue your project.

1.  **[Step 1: Deep Research](file:///.agent/workflows/.steps/vibe-step1.md)** - Analyze market opportunity and technical feasibility.
2.  **[Step 2: PRD (Product Requirements Document)](file:///.agent/workflows/.steps/vibe-step2.md)** - Define WHAT you are building.
3.  **[Step 3: Technical Design](file:///.agent/workflows/.steps/vibe-step3.md)** - Decide the best tech stack and implementation approach.
4.  **[Step 4: Generate Agent Instructions](file:///.agent/workflows/.steps/vibe-step4.md)** - Create AGENTS.md and agent_docs/.
5.  **[Step 5: Build MVP](file:///.agent/workflows/.steps/vibe-step5.md)** - Execute the build following the master plan.
